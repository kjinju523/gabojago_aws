# 🌐 VPC (Virtual Private Cloud)

## 1. VPC란?
- 사용자만의 **가상 네트워크** 환경
- AWS에서 리소스 간의 **격리된 네트워크 구성** 가능

## 2. 구성 요소
- **서브넷(Subnet)**: 퍼블릭/프라이빗 나눔
- **라우팅 테이블(RT)**: 경로 정의
- **인터넷 게이트웨이(IGW)**: 퍼블릭 인터넷 연결
- **NAT 게이트웨이**: 프라이빗 → 인터넷 허용

## 3. CIDR
- IP 대역 지정 (예: `10.0.0.0/16`)
- /16 → 65,536개 / /24 → 256개

## 4. 시험 팁
- 서브넷은 **AZ와 연결됨**
- 퍼블릭 서브넷: IGW + 라우팅 테이블 필요
- NAT Gateway는 **프라이빗 서브넷에서 외부로 나갈 때 사용**
